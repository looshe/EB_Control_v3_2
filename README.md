# EB Control v3.2 - Admin/Cadastros

## Login administrador
Usuário: admin
Senha: EB@Admin2026!

## Ajustes desta versão
- Menu Cadastros aparece para administrador.
- O usuário admin é tratado como administrador mesmo em bancos antigos.
- Gestão de usuários disponível em Cadastros.
- Criar usuário, alterar senha, ativar/desativar, tornar/remover admin e excluir usuário.
- Cadastro público pela tela de login permanece desativado.

## Como rodar
```bash
pip install -r requirements.txt
python app.py
```
Acesse: http://127.0.0.1:5000

Se trocar de versão, pare o servidor antigo com Ctrl+C e use Ctrl+F5 no navegador.
